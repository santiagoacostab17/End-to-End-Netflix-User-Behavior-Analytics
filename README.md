# Netflix User Behavior Analytics  
End-to-End Data Analysis & Predictive Modeling

Dataset: Netflix 2025 User Behavior (~210K records)

---

## Objective

Analyze user behavior to:

- Quantify engagement and churn
- Identify high-value user segments
- Detect churn risk drivers
- Build and evaluate predictive models
- Deliver reproducible analytics pipeline

---

## Workflow

1. Data validation & cleaning  
2. Feature engineering (behavioral & temporal features)  
3. Exploratory & statistical analysis  
4. KPI computation (Churn, Retention, ARPU, Engagement)  
5. Customer segmentation (K-Means)  
6. Predictive modeling (LogReg, Random Forest, Gradient Boosting)  
7. Model evaluation (ROC-AUC, Precision, Recall, F1)  
8. Model interpretation (feature importance, SHAP)  
9. Automated end-to-end pipeline  

---

## Key Insights (Sample)

- Engagement decay is a stronger churn predictor than total watch time  
- Device diversity positively correlates with retention  
- Early behavioral shifts signal churn risk weeks in advance
